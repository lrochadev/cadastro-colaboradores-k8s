# cadastro-colaboradores-k8s

**Kubernetes development tools**

* Docker
* Kubectl
* Minikube
* VirtualBox


`1 - minikube start`

`2 - minikube ip`

`3 - Mapear o ip do minikube: em: /etc/hosts para o host: dev.local`
    Exemplo: 192.168.99.100 dev.local

`4 - kubectl apply -f .`

`5 - Acesse o dashboard com o comando: minikube dashboard`
